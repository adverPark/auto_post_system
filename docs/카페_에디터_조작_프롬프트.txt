분석 결과를 바탕으로 카페 에디터 조작 코드를 만들어줘.

카페 에디터 특성:
- iframe 없음 (메인 페이지에 직접)
- 제목: textarea.textarea_input
- 등록: a.BaseButton.BaseButton--skinGreen (a 태그!)
- 소제목 서식 없음 → 인용구로 대체

HTML 본문의 각 태그를 이렇게 변환해줘:
- h2 → 인용구3 (bubble 스타일, 소제목 역할)
- h3 → 인용구2 (line 스타일)
- p → 일반 텍스트
- strong → 볼드
- ul/li → 앞에 "- " 붙여서 텍스트로
- table → 클립보드 붙여넣기 => html로 렌더링 해서 그 부분 복사해서 넣기.

먼저 테스트 HTML로 입력만 해보고, 발행은 하지 마.
