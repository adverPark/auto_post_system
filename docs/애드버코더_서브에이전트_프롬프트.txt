ADVERCODERAI_REST_API.md 파일을 읽고,
서브에이전트 생성 에이전트를 이용해서
애드버코더AI REST API로 블로그 글을 생성하는 서브에이전트를 만들어줘.

서브에이전트가 할 일:
1. 발행을 위한 블로그 목록 조회 (네이버블로그, 티스토리, 블로거, 네이버카페 등)
2. 제목+키워드 목록을 받아서 캠페인 생성 (INFO 또는 PARS 타입)
3. 발행처를 네이버 카페로 설정 (publish_status: "NCF")
4. 폴링으로 글 작성 완료 대기
5. 완료된 글 내용 수집
6. 발행 상태 확인 (카페 발행인 경우 pending 상태 확인)
7. 결과 반환 (글 제목, 상태, 발행 상태 포함)

기본 설정:
- 모델: GEMINI30FLASH
- 어투: FRD (친근한)
- API 키는 .env의 ADVERCODER_API_KEY 사용

titles 형식: "제목///키워드" (줄바꿈으로 구분)
