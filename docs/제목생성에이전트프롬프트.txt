keywords_expanded 상위 100개 키워드로 SEO 제목을 만드는 서브에이전트를 만들어줘.

## 자동완성 키워드 받는 방법
import requests, urllib.parse

keyword = "바이브코딩"
url = f"https://ac.search.naver.com/nx/ac?q={urllib.parse.quote(keyword)}&con=1&frm=nv&ans=2&r_format=json&r_enc=UTF-8&r_unicode=0&t_koreng=1&q_enc=UTF-8&st=100"
data = requests.get(url).json()

for item in data["items"][0]:
    print(f"  - {item[0]}")
바이브코딩 키워드로 자동완성을 받으니
  - 바이브코딩
  - 바이브코딩 뜻
  - 바이브코딩 책
  - 바이브코딩 앱
  - 바이브코딩 강의
  - 클로드 바이브코딩
  - 바이브코딩 ai
  - ai 바이브코딩
  - 바이브코딩 교육
  - 바이브코딩 러버블

이런 키워드를 받았다.


## 규칙:
- 키워드마다 네이버 자동완성 API 확인해서 관련 키워드 2~3개 뽑기
- 메인 키워드 + 자동완성 키워드로 제목 조합
- 모든 단어가 실제 검색 키워드여야 함
- "총정리", "꿀팁" 같은 비검색어 금지
- 15~25자, 메인 키워드 맨 앞
- 키워드 변형(조사/어미) 금지
- 나열식/슬래시 금지
- 자동완성에 없는 단어는 넣지 마
- 1 키워드 = 1 제목, 총 100개
- sonnet 모델 사용

결과는 blog_titles.json으로 저장해줘.
