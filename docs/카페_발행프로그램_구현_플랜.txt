플랜 모드로 시작해줘.

네이버 카페에 글을 자동 발행하는 프로그램을 만들어야 해.

요구사항:
1. 애드버코더AI REST API에서 pending 상태의 카페 포스트를 가져온다
   (GET /naver_cafe/posts/?cafe_id={cafeId}, Bearer 인증)
2. Playwright로 네이버 카페 글쓰기 페이지에 접속한다
3. 글 제목을 입력하고, HTML 태그별로 에디터 서식을 변환하여 본문을 작성한다
4. 발행 버튼을 클릭한다
5. 발행 완료 후 API로 상태를 업데이트한다
   (POST /naver_cafe/posts/update-status/)
6. 모든 pending 글을 순차적으로 처리한다

참고: ADVERCODERAI_REST_API.md 파일을 읽어서 API 구조 파악해줘.

네이버 로그인 방식:
- 첫 로그인: 프리셋에 아이디/비밀번호로 로그인 (프리셋에서 읽어서 하는 방식)
  (프리셋별로 아이디가 다르거나, 게시판이 다를 수 있음)
- 로그인 성공하면 쿠키(storage_state)를 JSON 파일로 저장
- 다음 실행부터는 저장된 쿠키로 로그인 스킵
(카페 발행 엘리먼트 검증시 로그인을 통해서 실제 직접 확인)
- 쿠키가 만료됐으면 다시 아이디/비밀번호 로그인
→ 이렇게 해야 테스트할 때마다 수동 로그인 안 해도 됨

검증은 단계별로:
- 로그인 먼저 구현해서, 먼저 코드에 직접 작성한 간단한 테스트 HTML로 에디터 동작 확인
- API 연동은 에디터가 확실히 되는 걸 확인한 후에
